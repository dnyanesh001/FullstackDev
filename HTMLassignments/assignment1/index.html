Projects

E-commerce Project

In my eCommerce project, I developed a full-stack application using the MEAN stack (MongoDB, Express.js, Angular, Node.js). The project involved creating a user-friendly interface for customers to browse and purchase products, as well as an admin panel for managing inventory, orders, and customer data. Key features included:

  • User Authentication: Implemented secure user login and registration using JWT.
  • Product Management: Developed CRUD operations for products, including image uploads and categorization.
  • Shopping Cart: Built a dynamic shopping cart with real-time updates and order processing.
  • Payment Integration: Integrated payment gateways for seamless transactions.
  • Testing: Conducted thorough testing using BDD Cucumber and automated tests with Java Selenium.

Banking Project

In my banking project, I worked on developing and automating various banking services. The project aimed to enhance the user experience and streamline banking operations. Key features included:

  • Account Management: Developed functionalities for account creation, balance inquiries, and transaction history.
  • Fund Transfers: Implemented secure and efficient fund transfer mechanisms.
  • Loan Processing: Automated loan application and approval processes.
  • API Testing: Utilized RestAssured for API testing to ensure reliability and performance.
  • Automation Testing: Employed Java Selenium, BDD Cucumber, Extent Reports, and TestNG for comprehensive testing.
  • STLC and SDLC: Applied Software Testing Life Cycle (STLC) and Software Development Life Cycle (SDLC) methodologies to ensure quality and timely delivery.
